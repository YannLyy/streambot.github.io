01

Introduction

Watch2Gether ("le service", "le bot") s'engage à protéger la vie privée de ses utilisateurs. Cette politique explique quelles données sont collectées, pourquoi elles le sont, et comment elles sont utilisées lorsque vous interagissez avec le bot Discord ou la plateforme web associée.

En utilisant Watch2Gether, vous acceptez les pratiques décrites dans cette politique.

02

Données collectées

Watch2Gether collecte uniquement les données strictement nécessaires au fonctionnement du service :

Donnée Source Usage Durée
ID Discord Discord OAuth2 Identification dans les sessions Session active
Nom d'utilisateur Discord Discord OAuth2 Affichage dans l'interface Session active
Avatar Discord Discord OAuth2 Affichage dans le chat Session active
Token JWT Généré par le service Authentification sécurisée 24 heures
Badges obtenus Activité dans le service Système de récompenses Persistant
Historique de session Participation aux sessions Timeline de visionnage Temporaire

Watch2Gether ne collecte jamais de mots de passe, d'emails, d'informations de paiement ou de données de localisation.

03

Comment nous utilisons vos données

Les données collectées sont utilisées exclusivement pour :

  • Vous identifier au sein d'une session de streaming collaborative
  • Maintenir votre authentification pendant la durée de la session (via cookie JWT, 24h)
  • Afficher votre nom et avatar dans le chat en temps réel
  • Synchroniser le contrôle de lecture entre les participants
  • Attribuer et afficher les badges liés à votre participation
  • Permettre la reprise de session pour les utilisateurs récents

Aucune donnée n'est utilisée à des fins publicitaires, analytiques externalisées ou de profilage comportemental.

04

Authentification Discord OAuth2

Watch2Gether utilise Discord OAuth2 avec le scope identify uniquement. Cela signifie que le service :

  • Peut lire : votre ID Discord, votre nom d'utilisateur et votre avatar
  • Ne peut pas accéder : à vos serveurs, vos messages, vos amis, vos emails ou tout autre donnée Discord
  • Ne stocke pas : votre token d'accès Discord — celui-ci est échangé immédiatement contre un JWT interne et n'est jamais conservé
05

Stockage et sécurité

Les données de session sont stockées en mémoire (RAM) sur le serveur et ne survivent pas à un redémarrage du service. Seules les données de badges sont persistées dans un fichier local sur le serveur hébergeant le bot.

  • Les tokens JWT sont signés avec une clé secrète et ont une durée de vie de 24 heures
  • Les cookies d'authentification sont transmis via HTTPS uniquement
  • Aucune base de données externe n'est utilisée pour stocker des données personnelles
  • L'accès aux routes protégées nécessite un token valide à chaque requête
06

Partage avec des tiers

Watch2Gether ne vend, ne loue et ne partage jamais vos données personnelles avec des tiers à des fins commerciales.

Le service interagit avec les services tiers suivants dans le cadre normal de son fonctionnement :

  • Discord API — pour l'authentification OAuth2 et les commandes slash
  • YouTube — pour la lecture des vidéos partagées (côté client uniquement, via le lecteur intégré)
  • Cloudflare Tunnel — pour l'exposition sécurisée du serveur web (aucune donnée stockée par Cloudflare au-delà du transit)

Ces services sont soumis à leurs propres politiques de confidentialité.

07

Conservation des données

Conformément au principe de minimisation des données :

  • Les données de session (ID, nom, avatar, JWT) sont supprimées automatiquement à l'expiration de la session (inactivité de 4 heures ou déconnexion)
  • Les tokens JWT expirent après 24 heures et ne sont pas renouvelables automatiquement
  • Les données de badges sont conservées de manière persistante tant que le service est actif, afin de préserver votre historique de participation
08

Vos droits

Vous disposez des droits suivants concernant vos données personnelles :

  • Droit d'accès : vous pouvez demander quelles données vous concernant sont traitées
  • Droit de suppression : vous pouvez demander la suppression de vos données (badges inclus)
  • Droit de rectification : si des données incorrectes vous concernent, vous pouvez en demander la correction
  • Droit d'opposition : vous pouvez cesser d'utiliser le service à tout moment pour mettre fin au traitement

Pour exercer ces droits, contactez-nous via les coordonnées ci-dessous.

09

Modifications de cette politique

Nous nous réservons le droit de modifier cette Politique de Confidentialité à tout moment. En cas de modification substantielle, la date de mise à jour en haut de cette page sera actualisée. L'utilisation continue du service vaut acceptation des modifications.

10

Nous contacter

Pour toute question, demande d'accès ou de suppression de données, vous pouvez nous contacter directement via Discord.
